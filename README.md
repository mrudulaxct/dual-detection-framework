Key Features

-> Dual Detection System: Two coordinated detectors (controller-side and plant-side)
-> Fault Detection: Detects system faults (sensor, actuator, plant failures)
-> Attack Detection: Detects stealthy cyber attacks (covert, zero-dynamics, replay attacks)
-> Anomaly Classification: Distinguishes between faults and attacks
-> Secure Communication: Encrypted network layer with AES encryption
-> Real-time Visualization: Live dashboard with detection statistics and system state
-> Database Logging: SQLite database for historical data storage
